Computer Vision, Machine Learning, Pytorch

ASL is my application that I creatdd for x, y, and z. ya ya ya d

Collecting Data

Using cvzone and Mediapipe, I implemented their hand landmark detection algorithm to identify hands in real-time. I wrote a script that allows saving a picture frame of the hand by pressing the 's' key. Each letter of the alphabet will be prompted multiple times (default is 10) to collect data for each letter.

Training

Data collected from the CV application will then be feed into a convolutional neural network (CNN). I used a CNN with 2 hidden layers, 64 outputs, and a kernel size of (3,3) and it seemed to work well. I switched to Google Colab for this part to generate the checkpoint/model because it is faster.

Testing & Refinement

Once the model is complete, run the model! See how accurate your model is to your data and refine.
